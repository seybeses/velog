<h2 id="💻-내-코드">💻 내 코드</h2>
<p>n=6명 -&gt; 6조각 (1조각씩) = 1판
10명 -&gt; 30조각 (5조각씩) = 5판
4명 -&gt; 12조각 (3조각씩) = 2판</p>
<pre><code class="language-python">import math

def solution(n):
    answer = 0

    def gdc(a,b):
        while b:
            a, b= b, a%b
        return a

    def lcm(a,b):
        return (a*b)/gdc(a,b)

    answer=lcm(6,n)/6
    return answer</code></pre>
<h3 id="💡-접근-방식">💡 접근 방식</h3>
<ol>
<li>n과 6의 최소공배수 구하기 (조각 수 구하기)</li>
<li>최소공배수 / 6 (판 수 구하기)</li>
</ol>
